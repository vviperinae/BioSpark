BioSpark Dashboard — Setup Instructions
========================================

1. Make sure Python is installed (python.org)

2. Install Flask:
   Open terminal/command prompt, type:
   pip install flask

3. Run the server:
   cd into this folder, then type:
   python app.py

4. Open dashboard:
   Open your browser and go to:
   http://127.0.0.1:5000

That's it! The dashboard will show all sensor readings.
